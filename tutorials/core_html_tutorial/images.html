Videos

Did you know you can also add video text tracks, like subtitles or transcript, to the video file. To do this we need to use the WebVTT format and add a track element inside the video element. This is how an example track element would look like: <track kind="subtitles" src="subtitles_es.vtt" srclang="es" label="Spanish" />
